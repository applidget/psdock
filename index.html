Overview

Psdock is a lightweight process launcher utility written in Go. It supports I/O redirection, log-rotation, and signal handling. Working on OS X and Linux, it can also be used as a replacement of /sbin/init inside containers. To give it a try, see Quickstart

Features

  • Redirection of the standard output & error to a file or a TCP connection
  • Standard input reading from a TCP connection
  • Configurable log-rotation
  • Specification of the child process execution context ( user, environment variablesâ€¦ )
  • TOML configuration file parsing

Quickstart

$ go get github.com/kr/godep
$ export PATH=$PATH:$GOPATH/bin
$ go get github.com/applidget/psdock
$ cd $GOPATH/src/github.com/applidget/psdock
$ make
